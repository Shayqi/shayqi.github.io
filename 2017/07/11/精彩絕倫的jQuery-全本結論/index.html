精彩絕倫的jQuery-全本結論

發表於 2017-07-11 | 文章點閱數 次  | 

第一章

分析了javascript熱門的第三方庫，為何從中選擇jQuery，振奮人心的開場，用最簡單的方式做最多的實現，
score: 9/10

第二章

安裝流程，說明了$(‘’)的jQuery包裝器，不需要用到$(document).ready(function(){})的地方–與DOM無關的語句，用jQuery別名防止第三方庫的衝突，試圖用jQuery取代javascript?
9/10

第三章

DOM樹選擇器，過濾器(:eq()/:gt()/:lt()/:not/:even、odd/:empty/:hidden/:visible/:has()…等，及基於jQuery選擇器所做的一些簡單範例
8.5/10

第四章

事件，有些部份過時了，例如unload事件只能以on(‘beforeunload’)方式調用，還有超越老套的bind()和live()上更方便的on方法，error也只能以on調用。此外，ready比之原生onload方法更快的原因在於DOM加載完後便執行ready，而不是等待全頁面加載完。需注意，預加載:load()方法加載完後便會直接插入DOM而非依照順序。
.hover事件合併了mouse的enter與leave，同時避免了over & out的事件冒泡問題。粗略帶過了:綁定、頁面、鼠標、鍵盤、表單的通用事件。
6/10

第五章

特效，講述了基本方法(show/hide/toggle/ slideDown/slideUp/slideToggle fadeIn/fadeOut/fadeTo)以及.delay(duration [,queue])延時效果，queue參數無詳解;
jQuery的鏈式調用，利用了end()返回操作對象的前一個對象，以及相當差的圖片相冊示例。
(搞了很久，示例a元素無法點擊原因在於被absolute元素所覆蓋，另，動畫插件要於options內設定easing值)
6.5/10

第六章

下拉菜單高亮示例的css設置錯誤(自修正)，折疊搭配精靈圖箭頭(header與icon各使用一個active類)。標籤nav參考源碼完成css配置，思路及架構說明還不錯，缺點是必須將頁面完成才知道要做哪些事&版面配置存在一些錯誤/缺少(汗)
5.5/10

第七章

利用表格建置美觀的數據展示，簡單說明了常見的條紋式、懸停式、crud操作，如何展示特價信息的方式，利用(網上抓的源碼)建構的分頁技術，主要是用了slice(start[,end])進行顯示數據的控制。
值得一提的兩個庫:tablesorter庫(表格排序)、visualize(表格可視化圖表、測試只有1.4.2版可讓jq套用，網上已建壯成商用)
6/10

第八章

表單-disabled屬性/ focus/blur方法 ，實現placeholder寫法，重複了一次第五章的字數限制示例….(全選與反選在checked屬性無論如何都會被勾選，必須使用內在特性.prop()設定)，驗證判斷三種結果(‘’,error,success)用正則.test匹配正確值，.val()獲取表單值，複製技巧-將值保存後利用事件(點擊)賦值。
qtip庫(tooltip)書上已過時，測試了3.0.3版，的確相當好用。(註:作者的label標籤一直用錯…)
validate庫，添加required類，說明了一些基本規則用法，對於需要驗證多數據表單應該不錯。
5.5/10

第九章

Ajax，基本.load(url[,data][,callback(responseText,textStatus,XMLHttpRequest)]))的用法，響應碼通過XMLHttpRequest.status傳給瀏覽器。
常見響應碼:{
  200:成功;
  301:永久跳轉; 302:臨時跳轉;
  400:錯誤請求; 401:未授權 403:禁止訪問 404:未找到
  500:服務器錯誤
}url內也可載入指定的class/id/tag name
get，post方法說明，.利用ajax([settings])載入內容，介紹了一個delicious API..此網站已轉行
跨域傳輸JSON數據的JSONP技術，練習到yelp要用post方法獲取訪問令牌太過麻煩，暫略…
老掉牙的補充說明:基本的ajax–jsonp技術是使用函數包裝json資料，以url後+callback方式載入，而jQuery的.ajax()已打包好這一切，在datatype:’jsonp’設定即可。
4/10

第十章

jQueryUI方法現在來看已頗不合時宜，但主要還是介紹了幾個常用組件，tabs/accordion/autocomplete/datepicker，及更小的UI庫-jQuery Tools(包含6組件:tabs/tooltips/overlays/scrolling/forms/toolboxes)目的為簡化以提升效能，但不包括css，改造或優化再做考慮，此處略…
還有類似於lightbox的輕量庫-fancybox介紹，可以將圖片層從頁面中提出展示。
自寫插件，書中有不少疏漏，所幸有抓源碼…
發布插件三步驟:
  1.測試是否兼容所有瀏覽器
  2.README 說明選項含義及工作原理
  3.選擇開源許可協議
打包插件: 若源碼<4kb則略，否則建議使用代碼壓縮器最小化
可發布站點 : plugins.jquery.com  &&   code.google.com/hosting
6/10

第十一&十二章

用Apache的Mod_rewrite模塊重定向用戶瀏覽器，用以檢查各種移動設備…
再談jQuery Mobile、其它移動框架，現已十分強大的如react_Native，ionic，還有書中提的sencha touch是從jQtouch而來，也可供參考。
還有一些jQuery資源匯整，論壇、大會等…
8/10

全書總評

說明足夠白話，但深入理解不足，大多需靠自已補充(大推jQuery api 中文文檔)，但示例易懂，尚不失精采。
avg: 6.73/10
